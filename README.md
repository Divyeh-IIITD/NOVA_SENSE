WEEK 2 - learnt xgboost, catboost, hgboost but lgbm still gave highest accuracy
WEEK 3 - implemented SHAP analysis on lgbm
WEEK 4 - implemented user interface 
WEEK 5 - polished the Streamlit demo and wrapped up the project for now
Run the Streamlit demo from the project root with `run_streamlit.bat`, or use `python -m streamlit run streamlit_app.py`.
